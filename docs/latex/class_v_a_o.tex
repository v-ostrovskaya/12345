\doxysubsection{Detailed Description}
Vertex Array Object (\doxyref{VAO}{p.}{class_v_a_o}) class. 

\doxyref{VAO}{p.}{class_v_a_o} is used by Open\+GL to store Vertex Buffer Objects (\doxyref{VBO}{p.}{class_v_b_o}) and Element Buffer Objects (\doxyref{EBO}{p.}{class_e_b_o}) in one object, making it easier to bind them.
