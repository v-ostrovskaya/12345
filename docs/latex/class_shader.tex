\doxysubsection{Detailed Description}
This class manages shaders. 

Shaders are programs that take vertices as input and eventually determine how pixels are rendered on the screen. We are using a vertex shader to transform \doxyref{VAO}{p.}{class_v_a_o}\textquotesingle{}s vertices using matrices and a fragment shader to change the vertex color.
